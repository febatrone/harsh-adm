# Harsh Thaker — Portfolio

A high-impact electronics and design portfolio built with React, TanStack Router, and Tailwind CSS.

## Features

- **Interactive Shader Background:** WebGL-powered dynamic "waves" in the Hero section.
- **Typographic Precision:** Optimized font scale for readability and impact.
- **Glassmorphic Design:** Modern, high-contrast UI with smooth animations.
- **Fully Responsive:** Mobile-first design that scales beautifully to desktop.

## Local Development

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd enchanted-canvas
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## Tech Stack

- **Framework:** React 19
- **Routing:** TanStack Router
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Epilogue (Headlines), Plus Jakarta Sans (Body), Geist Mono (Mono)
